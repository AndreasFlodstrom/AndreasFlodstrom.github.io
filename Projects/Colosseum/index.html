About the Project

Colosseum is a small 2D combat platformer where the player fights against a boss through three distinct stages, each increasing in difficulty. The game focuses on smooth mechanics, tight movement, and reactive boss behavior to create a fast-paced and satisfying gameplay loop.

Project Info

  • Duration: 2 months
  • Technologies: C#, Unity, Blender
  • Role: Boss combat system designer, 3D asset helper
