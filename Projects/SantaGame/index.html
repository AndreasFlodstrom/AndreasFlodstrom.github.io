Project: Santa

3D Co-op Party Game

About the Project

Project: Santa is a 3D co-op multiplayer game inspired by Overcooked. Players work together—or against each other—to assemble as many toys as possible before time runs out.

Deep Dive

This project was created as a personal challenge to push my game development skills. I designed all aspects of the game—from gameplay logic to 3D modeling in Blender. Networking is powered by Mirror for multiplayer support. SteamSDK integration provides achievements and online play.

Project Info

  • Duration: 3 months
  • Technologies: C#, Unity, Blender, Mirror, SteamSDK
  • Role: Full game development (design, programming, 3D art, networking)
