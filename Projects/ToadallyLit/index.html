Toadally Lit

2D Platformer Combat Game

About the Project

Toadally Lit (originally Colosseum) is a 2D platformer combat game focused on responsive controls and challenging boss battles.

Development Details

This project taught me Unity fundamentals including physics interactions, sprite animation, and enemy AI design. Blender was used for 3D assets exported into the 2D environment to enhance visual depth.

Project Info

  • Duration: 2 months
  • Technologies: C#, Unity, Blender
  • Role: Boss combat system designer, 3D asset contributor
